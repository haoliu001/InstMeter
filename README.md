## Overview

**InstMeter** is an instruction-level method designed to predict the **energy consumption** and **latency** of deep learning (DL) model inference on **microcontroller units (MCUs)**.  
This repository aims to provide implementations, scripts, and (future) tooling to reproduce the results presented in the paper.
